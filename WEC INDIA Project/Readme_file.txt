Welcome to the World Energy Council (WEC) India Project Repository! This repository contains two interactive Power BI dashboards that provide in-depth insights into India's electricity consumption trends and projections. These dashboards are part of ongoing efforts to support informed decision-making in the energy sector by presenting actionable data in an easy-to-understand format.

The two dashboards available in this repository are:

1.India's Yearly Electricity Consumption Analysis 2.India's Electricity Sales vs EPS 19 Analysis

These dashboards leverage a range of data sources to offer insights into electricity consumption patterns, sales projections, and state-wise deviations.
